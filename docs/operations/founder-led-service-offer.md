# ScopeHouse Founder-Led Service Offer v1

## Purpose

Define the first service offer ScopeHouse should sell before broad self-serve SaaS launch.

This offer exists to:

- create early revenue
- validate real buyer pain
- collect live project artifacts
- teach the product through founder-led setup
- build case studies and stronger onboarding insight

## Offer name

ScopeHouse Project Setup

## Short description

A founder-led service for homeowners and small renovation pros who need help turning a messy renovation into a clean project record with clearer scope, quote comparison, decisions, and next steps.

## Core positioning

This is not abstract consulting.

This is practical project organization using the ScopeHouse workflow.

The buyer is paying for:

- faster clarity
- cleaner project records
- less quote confusion
- stronger decision tracking
- a project that feels under control sooner

## Target buyer

### Primary buyer now

Homeowners planning medium to large renovations who already feel project confusion before work is fully under control.

Best fit signals:

- multiple rooms or phases
- active quote collection or pre-construction work
- scattered notes, spreadsheets, emails, and texts
- family or contractor coordination friction
- visible cost attached to unclear scope or decisions

### Secondary buyer

Solo GCs, renovation consultants, and small design-build operators who want a cleaner project control layer without heavy contractor software.

## Main problem this solves

The project is real, but the record is weak.

The buyer has:

- unclear scope
- poor quote comparison
- scattered documents and notes
- missing decision history
- weak change visibility
- no single source of truth

## Core deliverable

By the end of the service, the client should have:

- one structured project record
- a cleaner working scope
- a clearer view of budget and project frame
- organized notes, documents, and decisions
- a summary they can review and share

## Service tiers

### Tier 1, Light setup

Price range:

- $300 to $500

Best for:

- smaller but still meaningful projects
- users early in planning
- users who need structure but not deep quote cleanup

Includes:

- intake review
- project setup in ScopeHouse
- basic scope organization
- one summary output
- one follow-up session

### Tier 2, Core setup and quote comparison

Price range:

- $600 to $900

Best for:

- active renovation planning
- users collecting or reviewing quotes
- users with scattered records and visible confusion

Includes:

- intake review
- project setup in ScopeHouse
- scope cleanup
- quote comparison support
- decision log setup
- project summary output
- one to two follow-up sessions

This should be the main offer used first.

### Tier 3, Deep setup

Price range:

- $1000 to $1500

Best for:

- larger or more complex renovations
- more stakeholders
- repeated working sessions
- heavier project cleanup needs

Includes:

- deeper setup and organization
- more involved scope cleanup
- quote and decision review across multiple sessions
- stronger project record handoff
- more follow-up support

## Recommended starting offer

Start with Tier 2 as the default sell.

Why:

- strongest link to current product wedge
- easiest to explain
- strongest visible value
- good middle price anchor
- enough depth to expose real workflow pain without turning into open-ended consulting

## Delivery steps

### Step 1, qualification

Confirm:

- active project timing
- scope confusion
- quote comparison pain
- real decision maker involvement
- fit with ScopeHouse workflow

### Step 2, intake collection

Collect:

- project goals
- rooms or phases
- constraints
- budget range
- current quotes if available
- current notes or docs if available

### Step 3, setup session

Create the project record with the client.

Do:

- guided intake
- structure the project
- define current project frame
- identify major uncertainty points

### Step 4, cleanup and organization

Do:

- clean up scope language
- organize quote comparison if present
- set up decisions and key notes
- capture immediate next steps

### Step 5, review session

Walk the client through:

- updated project record
- main gaps and risks
- cleaner project summary
- what to do next

### Step 6, handoff

Provide:

- live ScopeHouse project record
- exportable summary
- short next-step recommendation

## Boundaries

Do not let this become:

- general contractor management
- unlimited consulting
- procurement management
- accounting support
- architectural or engineering advice
- legal review of contracts

The service should teach the ScopeHouse workflow, not replace it with custom manual work.

## Offer language for selling

### Simple version

I help you turn a messy renovation into a clear project record so you can define scope, compare quotes, track decisions, and move forward with less confusion.

### Slightly more direct version

If your renovation is spread across notes, texts, quotes, and memory, I can help set it up in ScopeHouse and turn it into a cleaner working plan.

## Objections this offer should handle well

### I do not want more software

Response direction:

This is not a DIY software pitch first. I help set the project up with you and leave you with a cleaner system.

### I already have notes and spreadsheets

Response direction:

That is exactly the point. This service helps turn scattered records into one working project view.

### I am not sure the project is far enough along

Response direction:

This works best before confusion gets more expensive. Early planning is a good time to clean up the record.

## Success criteria

This offer is working if:

- it closes faster than pure software pitches
- users show up with real project artifacts
- clients understand the value of one project record
- the work produces reusable product insight
- some buyers convert into longer-term product users or referrals

## Operational notes

Use this offer in:

- discovery follow-up
- design partner qualification
- paid-service routing from the tracker
- landing page experiments
- warm referral conversations

## Recommendation

Sell one named offer first.

Do not start with multiple service packages in public.

Publicly lead with:

- ScopeHouse Project Setup

Internally use three service depth levels for pricing flexibility.
